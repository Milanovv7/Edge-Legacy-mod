# Edge-Legacy-mod
Edge Legacy mod is designed to mimic the look of the old Microsoft Edge. The Microsoft Edge from early Windows 10 builds like 1507 and 1607. To get it, Make sure you have Chrome installed on your computer. Then, in a new tab search for "Chrome Web Store" and click on the first link. Download the Ignore X Frame headers extension. Then click on https://milanovv7.github.io/Edge-Legacy-mod/ and click on the 3 lines in the toolbar (near the top right corner), Cast, Save and Share and then install page as app. Read the OS requirements at https://github.com/Milanovv7/DragonWeb-Browser. The OS requirements for DragonWeb are the same for Edge Legacy Mod.
